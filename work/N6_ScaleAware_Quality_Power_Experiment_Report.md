# N6-A 分支自适应质量指数实验报告

## 1. 实验目的

验证 P3 使用较低质量指数、P4/P5 保持 `power=3.0`，能否保留 P3 小目标候选和 Recall，同时维持 N4-QP3 的高 mAP50-95。

## 2. 实验设计

复用 N4 seed 0 和 C1A seed 0 的既有权重，仅修改推理阶段 P3/P4/P5 的质量指数：

```text
[3,3,3], [0,3,3], [0.5,3,3], [1,3,3], [1.5,3,3], [2,3,3]
```

代码在每个尺度的质量分数拼接前独立执行：

```text
score_l = class_score_l * quality_score_l ** power_l
```

该实验不重新训练，不改变参数量、GFLOPs 或损失函数。

## 3. 实现验证

`[3,3,3]` 与原标量 `power=3.0` 完全一致：

| 模型 | 标量 QP3 | 分支 `[3,3,3]` |
|---|---:|---:|
| N4 mAP50-95 | 0.678852 | 0.678852 |
| C1A mAP50-95 | 0.677000 | 0.677000 |

说明分支顺序和实现正确。

## 5. 判定

YOLO11n seed 0 为 `R=0.778`、`mAP50=0.844`、`mAP50-95=0.657`。没有候选同时满足：

```text
Recall > 0.778
mAP50 >= 0.844
mAP50-95 > 0.657
```

也没有候选满足预设 A 级条件 `R>=0.760` 且 `mAP50-95>=0.674`。

## 6. 现象分析

降低 P3 power 能提高 Recall，但 mAP50-95 同步大幅下降。C1A `[0,3,3]` 的 Recall 达到 0.774499，距离 baseline 仅 0.003501，但 mAP50-95 降至 0.643827；N4 `[0,3,3]` 的 mAP50-95 仅为 0.639006。

原因不是 P3 缺少候选，而是跨尺度分数失配：P3 使用低 power 后，其质量因子整体高于采用 QP3 的 P4/P5，导致 P3 候选在全局排序和 NMS 中被相对放大。新增低质量候选提高 Recall，却降低了高 IoU AP。

C1A 在低 P3 power 下通常比 N4 获得更高 Recall、mAP50 和 scallop AP，说明 MSFF 确实增强了 P3 候选；但该增益不足以同时超过 baseline 和 N4-QP3，因此不进入多种子训练。

## 7. 结论

```text
N6-A 分支独立 power：失败
MSFF 多种子训练：停止
P3/P4/P5 power 搜索：停止
当前最优主模型：N4-QP3
```

分支独立指数不能直接解决 Recall 与 mAP50-95 的权衡，并引入跨尺度置信度不可比问题。

## 8. 下一实验

下一步验证全尺度一致的残差质量融合：

```text
score = class_score * ((1 - lambda) + lambda * quality_score ** 3)
```

所有尺度使用相同 `lambda`，避免新的跨尺度分数偏移。先复用 N4 seed 0 权重做免训练扫描；若仍无法形成 Recall 与 mAP50-95 的 Pareto 改善，则停止后处理校准路线，转向训练阶段的质量监督改进。

## 9. 论文价值

该负结果可用于说明为什么本文不采用简单的尺度独立超参数：水下小目标虽然需要较弱质量抑制，但未经分布对齐的分支独立指数会破坏跨尺度置信度可比性。若后续提出带归一化或残差约束的质量融合，该实验可以作为其设计动机和消融对照。
